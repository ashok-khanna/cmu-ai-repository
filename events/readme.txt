This directory holds announcements of events of interest to AI researchers.

   meetings/	Announcements of conferences, workshops, and symposia.
	        Summarized in the file SUMMARY.
   contests/	Announcements of AI-related contests.
   journals/	Announcements of journal special issues.

Please send submissions to mkant+ai-events@cs.cmu.edu. If sending a meeting
announcement, please include an entry for the SUMMARY file.
