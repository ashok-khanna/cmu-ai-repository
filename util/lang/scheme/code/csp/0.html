Constraint: Port of Steele's Constraint System

lang/scheme/code/csp/
A translation of the constraint system first published in Steele, Guy Lewis Jr., "The Definition and Implementation of a Computer Programming Language Based on Constraints", MIT Artificial Intelligence Laboratory TR-595, August 1980. The translation is written for the current distribution of MIT Scheme.
