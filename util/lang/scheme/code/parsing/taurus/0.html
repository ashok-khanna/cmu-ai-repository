Taurus: A recursive descent parser generator for Scheme.

lang/scheme/code/parsing/taurus/
Taurus is a parser generator written in Scheme. It generates a recursive descent parser from a LL(1) grammar augmented with semantic actions supplied by the user. The resulting parser runs in Scheme, and can perform automatic syntactic error recovery.
