AVL: AVL trees and tries.

lang/scheme/code/ext/avl/
AVL trees are balanced binary trees. This implementation recreates the tree around the inserted element, without side effects.
Version: AVL trees 1.1 (27-OCT-93); AVL tries 1.1 (4-DEC-93) Ports: Tested in SCM. Should run in R4RS compatible schemes with minor modifications.
