Pass2: Second pass of the Scheme 313 compiler

lang/scheme/code/eval/pass2/
This directory contains code for the second pass of the Scheme 313 compiler, including single assignment analysis, assignment elimination, lambda lifting, and various local source transformations. It operates as a source-to-source transformation on a subset of Scheme.
