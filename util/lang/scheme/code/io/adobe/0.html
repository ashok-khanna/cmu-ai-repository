Adobe: Portable 2d gray-scale graphics package for Scheme.

lang/scheme/code/io/adobe/
ADOBE is John Bertoldi's portable 2d gray-scale graphics package for Scheme. It implements an abstract graphic device, which, when written to, produces a file that can be read by Adobe Illustrator (TM) and most other page composition and illustration-editing applications. Based on a similar, earlier module by O. Schlageter. Includes documentation.
