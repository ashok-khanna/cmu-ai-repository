Syntax-Case Macros: Proposals and implementations for macros based on syntax-case.

lang/scheme/doc/macros/sc/
This directory contains proposals and implementations for macros based on syntax case. It includes a copy Robert Hieb and Kent Dybvig's implementation and two related technical reports [6, 8].
