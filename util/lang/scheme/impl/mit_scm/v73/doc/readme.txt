This directory contains documentation for the 7.3 release.  All of the
files in this directory are compressed with GNU gzip.

    info/*		Info files for Emacs or Edwin
    refman.dvi		DVI file of printed document
    refman.ps 		PostScript file of printed document
    refman.texinfo	Texinfo source for document
