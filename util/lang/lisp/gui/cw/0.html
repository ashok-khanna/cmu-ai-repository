Common Windows: Information about Allegro Common Windows

lang/lisp/gui/cw/
Allegro Common Windows provides a front end to CLX. It is *not* free. Contact info@franz.com for more information. Intellicorp's KEE4.0 also comes with an implementation of Common Windows. They've implemented the CW spec to run on Lucid 4.0 on Sparcs, HP300/400s, HP700/800s, and IBM RS6000s. Contact tait@intellicorp.com for more information.
