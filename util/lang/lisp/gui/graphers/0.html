Graphers: Simple grapher that runs in most of the GUIs

lang/lisp/gui/graphers/
This directory contains a simple grapher like the one described in "Lisp Lore" by Bromeley and Lamson. It has versions for most of the GUI toolkits. Several implementations have a mouse sensitivity feature and others have implementation-specific features.
