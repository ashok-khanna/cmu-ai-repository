SciGraph: BBN's Scientific Graphing Package

lang/lisp/gui/clim/general/scigraph/
Scigraph (Scientific Graphing Package) is a an object-oriented library for doing x-vs-y plots. It can be embedded in other applications. Scigraph's functionality includes scatter plots, linear plots, bar charts, contour plots, and symbolic equations. Scatter, linear, and bar plots are fast and reliable. Graphs and data points are mouse-sensitive. You may either interactively or programmatically control zooming, annotating, setting colors and symbols, and the usual features you would expect.
