MINIMAX: Lisp code for alpha beta minimax search.

lang/lisp/code/search/minimax/
Common Lisp implementation of alpha-beta minimax search by Mark Kantrowitz (18-NOV-90). Includes a NEG-MAX implementation. A simple implementation of alpha-beta search for tic-tac-toe by Marco Zagha at Carnegie Mellon University for "AI: Representation and Problem Solving" (15-381), Spring 1990.
