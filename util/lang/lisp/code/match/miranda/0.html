Select Match

lang/lisp/code/match/miranda/
The SELECT-MATCH macro is useful for implementing interpreters, (defun eval-expr (x) (select-match x ('add x y) => (+ x y) ('sub x y) => (- x y) ('mul x y) => (* x y) ('div x y) => (/ x y))) and also for programs in general, (defun my-append (a b) (select-match a 'nil => b (hd . tl) => (cons hd (my-append tl b)))) (defun fact (n) (select-match n '0 => 1 n => (* n (fact (1- n)))))
