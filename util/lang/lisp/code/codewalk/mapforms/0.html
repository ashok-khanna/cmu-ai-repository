Mapforms: Moon's code walker

lang/lisp/code/codewalk/mapforms/
Moon's MAPFORMS program is a tool for walking over your lisp source code. It can do substitutions as well as analysis (most of the complexity comes from the goal of being able to make substitutions with minimum consing).
