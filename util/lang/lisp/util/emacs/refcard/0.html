Reference card for Allegro CL and Franz's Emacs interface.

lang/lisp/util/emacs/refcard/
This directory contains TeX source for a quick and dirty Franz+Emacs reference card, closely modeled on the GNU Emacs reference card. It summarizes function and variable names in Franz Allegro 4.1, and key bindings for the emacs interface.
