ROLOG: Parallel Prolog compiler with a reduce-OR process model.

lang/prolog/impl/parallel/rolog/
ROLOG is a parallel PROLOG compiler with a reduce-OR process model. It is machine independent, and runs on both shared and nonshared memory machines. Example programs include N-Queens, Towers of Hanoi, and Quicksort.
