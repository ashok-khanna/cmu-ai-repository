I have finished the first version of ProTcl, a Prolog interface to Tcl/Tk.
It can be obtained via anonymous ftp from ecrc.de, in /pub/eclipse/progs/protcl.
It is a rather simple interface, all it does is to allow to evaluate
Tcl expressions from Prolog and (in some Prolog systems), to have the interactive
functionality of wish, i.e. one can type Prolog or Tcl queries and at the same
time X events are served. It has been tested on ECLiPSe and SICStus,
on a Sparc 10. Comments and suggestions please to me.

--Micha
