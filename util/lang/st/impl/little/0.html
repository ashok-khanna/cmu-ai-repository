Little Smalltalk: Tim Budd's version of Smalltalk

lang/st/impl/little/
This directory contains Tim Budd's implementation of Smalltalk. The windowing interface is built on top of Guido van Rossum's standard window package, and runs on top of systems that support this package, including X-11 and the Macintosh.
