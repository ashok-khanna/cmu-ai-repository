AGENT_CLIPS: Multi-agent tool for the Macintosh

areas/expert/systems/clips/agent/
AGENT_CLIPS is a multi-agent tool for Macintosh. Multiple copies of CLIPS runs at the same time on the Macintosh. Each Agent (CLIPS) can send commands to other active agents at run time. AGENT_CLIPS handles incoming commands automatically. Command transfer also means that you can exchange facts and rules at run time. This is a form of Knowledge Exchange among Intelligent Agents. Version 1.0 is free for introduction. Later Versions of AGENT_CLIPS might NOT be free. Source code is not available.
