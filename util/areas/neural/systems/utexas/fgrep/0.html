FGREP: Distributed representations for words in the case-role assignment task.

areas/neural/systems/utexas/fgrep/
This package contains the code and data for nonrecurrent FGREP simulation in the case-role assignment task, i.e. in mapping sentence constituents into case roles, simultaneously developing distributed representations for the words.
