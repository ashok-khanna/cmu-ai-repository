In this directory we have collected some routines and programs
that are sometimes useful with SOM_PAK and LVQ_PAK programs.
These programs have been made by various people in various sites,
so no coherent programming style can be expected. 

Currently there are the following programs:

Xvisual.c:   Is like the gvisual-program in SOM_PAK, but
             programmed with X11R5-calls. No references
             to GL-calls. See README.Xvisual and makefile.Xvisual.
