Blue: Backpropagation and Conjugate Gradient programs in Fortran

areas/neural/systems/blue/
This directory contains the backpropagation (bp) and conjugate gradient (cg) programs written by James Blue. Both programs train and test standard "3-layer" fully-connected feed-forward networks and are written in Fortran using the Ratfor preprocessor.
