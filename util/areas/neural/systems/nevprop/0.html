NevProp: General purpose backpropagation in C.

areas/neural/systems/nevprop/
NevProp is a C implementation of general purpose backpropagation software, based on Quickprop 1.0 by Scott Fahlman, as translated from Common Lisp into C by Terry Regier. The quickprop algorithm itself has not changed substantially, but it now includes options to force gradient descent (per-epoch or per-pattern), generalization & stopped training, c index, and interface enhancements. Questions and comments regarding the Quickprop algorithm itself should be directed to Scott Fahlman at sef+@cs.cmu.edu. Issues pertaining to NevProp code or functionality itself should be directed to the authors.
