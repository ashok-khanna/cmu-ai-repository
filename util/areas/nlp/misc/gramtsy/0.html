GRAMTSY: GRAMmatical Transformational SYstem

areas/nlp/misc/gramtsy/
This directory contains GRAMTSY, an interpreter for transformational grammars that work on labeled bracketings. GRAMTSY accepts grammars in a more or less linguistically familiar notation, and applies them in a straightforward way.
