LDB: Demo version of the Linguistic DataBase program

areas/nlp/misc/ldbdemo/
The Linguistic DataBase (LDB) program is a database system for syntactic analysis trees. The LDB program, together with a 130,000 word analyzed corpus of modern English (the Nijmegen corpus), is available free of charge to academic institutions andfor Hfl. 5000,- to others. This directory contains a demo version of the LDB program.
