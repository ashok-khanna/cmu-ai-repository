BABBLER: Simple Markov chain language acquisition-and-use

areas/nlp/misc/babbler/
This directory contains BABBLER, a program that a keystroke stream "sliding window" as an echoic (short term) memory and stores overlapping replicas of the successive windows in an associative memory which is driven by a Markov chain to create a language acquisition-and-use system. A compiled executable for IBM PCs is available for $34 from the author.
