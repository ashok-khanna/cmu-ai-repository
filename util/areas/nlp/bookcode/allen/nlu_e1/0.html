areas/nlp/bookcode/allen/nlu_e1/
This directory contains software from James Allen, "Natural Language Understanding", The Benjamin/Cummings Publishing Company, Inc., 1987. ISBN 0-8053-0330-8. (First edition) and is intended to be used with the book. It includes a simple RTN (recursive transition network), ATN (augmented transition network), an Eliza program, and a logic-based parser.
