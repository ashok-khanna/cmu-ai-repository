This directory contains Mark Kantrowitz's BibTeX bibliography of research
in natural language generation. When appropriate, please cite the technical
report, 
   Mark Kantrowitz, "Bibliography of Research in Natural Language
   Generation", Technical Report CMU-CS-93-216, Carnegie Mellon
   University, November 1993.

Please note that a customized version of bibtex was used to generate the
technical report, part due to the large size of the bibtex file. This shouldn't
affect most users of the bibliography, however, since the unmodified version
of bibtex works fine for small numbers of citations.

If you have any questions, please contact mkant@cs.cmu.edu. Contributions are
always welcome.
