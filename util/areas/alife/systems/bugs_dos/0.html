BUGS: Critters hunt bacteria.

areas/alife/systems/bugs_dos/
This directory contains a graphic simulation of Darwinian evolution in which "bugs" learn to hunt "bacteria". It is based on an idea from A.K. Dewdney's Scientific American article.
