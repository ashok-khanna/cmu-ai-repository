Conway's Game of Life for VT100 using curses/termcap.

areas/alife/systems/life/rupley/
This directory contains an implementation of Conway's Game of Life for VT100 terminals using curses or termcap. Includes a large collection of Life patterns. This program is based on an old implementation by Leor Zolman for the 8080; a copy of Zolman's original code is also included.
