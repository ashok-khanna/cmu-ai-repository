Soar: Integrated Agent Architecture

areas/planning/systems/soar/soar5/
This directory contains the Lisp version of Soar. Soar is an integrated agent architecture that supports learning through chunking. Included are CParaOPS5, TAQL (Task Acquisition Language), soar-mode for GNU Emacs, the SoarX graphic display, and SoarSIM (simulation building tool for Soar).
