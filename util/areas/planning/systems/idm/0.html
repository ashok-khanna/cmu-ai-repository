IDM: STRIPS-like planning

areas/planning/systems/idm/
This directory contains IDM, a STRIPS-like planning system. Examples include standard Blocksworld problems, Towers of Hanoi, and the Register-Swapping Problem. Also included is a C implementation of sockets with a Lisp interface and a Lisp implementation of distributed task allocation for load balancing. (?)
