Hello

I have placed an updated version of Spiro Michaylov's bibliography on
Constraint Logic Programming in the comp.constraints ftp area. 

	ftp.cs.city.ac.uk [138.40.91.9],
	pub/constraints/clp2.{bib,tex,ps}

Also accessible via the World Wide Web:
http://web.cs.city.ac.uk/archive/constraints/constraints.html

Run Latex once on clp2.tex. Then run Bibtex once on clp2.bib. (You
should ignore 11 warning messages.) Then run Latex TWICE on clp2.tex.

The postscript version is there for people who don't have Latex.

Good news: there are 720 entries, approximately double the previous
number.

Bad news: the key format is not the same as Spiro's. Sorry.

Any errors etc, please tell me. Blame me not Spiro.
