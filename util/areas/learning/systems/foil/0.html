This directory includes several versions of Quinlan's FOIL (First Order Inductive Learning) machine learning program.
References: Quinlan, J.R. (1990), "Learning Logical Definitions from Relations", Machine Learning 5, 239-266. Quinlan, J.R. (1991), "Determinate Literals in Inductive Logic Programming", Proceedings 12th International Joint Conference on Artificial Intelligence, 746-750, Morgan Kaufmann.
