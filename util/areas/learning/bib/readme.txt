This directory contains BibTex files originating from the ILP
ESPRIT project.

________________________________________________________________

- lrefs.bib.Z
BibTex file developed by Luc de Raedt and Stephen Muggleton for 
a Logic Programming Journal article on ILP (around 325 entries)

- ILPBook.bib.Z
BibTex file, which Nada Lavrac created for the forthcoming book
"Inductive Logic Programming: Techniques and Applications",
published by Ellis Horwood (around 200 entries).
