AGENT0 Interpreter: Agent-Oriented Programming

areas/agents/aop/
This directory contains an implementation of AOP, an interpreter for programs written in AGENT0. AGENT0 is a programming language based on Yoav Shoham's Agent-Oriented Programming Paradigm. AOP includes a nice GUI based on MCT (Mark's CLX Toolkit). A demo of an airplane agent is included.
