


ELIZA Ver 2.5

by Tom Bender of TRansTex Software, Aug 4, 1992.

Downloaded from UMich Archive for the Mac by Nena Marin ME391R student Summer 93.

Eliza is the classic psychologist-in-a-box AI simulation.  This is a clever Text editor with built in psychologist.
This version will talk if MacinTalk is installed.  A demo is included.
