Prolog implementation of Eliza

areas/classics/eliza/prolog/
This directory contains several implementations of Eliza in Prolog. uga.tgz contains eliza.ari and eliza.pl, an implementation by Viren Patel for Arity Prolog and Quintus Prolog, respectively. njr.tgz contains an implementation by Neil J. Rubenking
