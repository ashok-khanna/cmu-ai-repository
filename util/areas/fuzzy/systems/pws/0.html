This directory contains demo versions of PWS Publishing Company's "FUZZY SYSTEMS TOOLBOX FOR USE WITH MATLAB(r)". A UNIX version will be available in the near future. The demo may also be obtained on floppy disk by calling ITP Faculty Support, 1-800-423-0563, and asking for demo package ISBN#: 0-534-94593-7.
