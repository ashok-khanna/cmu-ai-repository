TIL2MSG: Converts Fuzzy-C Compiler or TILGen messages

areas/fuzzy/systems/til2msg/
This directory contains til2msg.exe, a filter program that converts the messages from the Fuzzy-C Compiler (FCC) or TILGen into a format that is displayable in Borland's C++ IDE.
