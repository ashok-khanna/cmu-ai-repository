SGA-P: Goldberg's Simple Genetic Algorithm Pascal

areas/genetic/ga/systems/sga/sga_p/
This directory contains Goldberg's Simple Genetic Algorithm, in Turbo Pascal and Sun Pascal. It was typed in by Gary Gibson, based on the code in Appendix C of Goldberg's book (minus the comments), and then made more portable by Gibson and Heitkoetter.
