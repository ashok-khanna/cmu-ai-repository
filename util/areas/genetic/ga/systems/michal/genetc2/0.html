Genetic-2: Optimization package for the linear transportation problem

areas/genetic/ga/systems/michal/genetc2/
This directory contains the Genetic-2 genetic algorithm numerical optimization package by Zbigniew Michalewicz, as described in his book. Genetic-2 is an optimization package for the linear transportation problem (minimization of the transportation cost).
