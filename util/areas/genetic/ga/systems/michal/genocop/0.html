GENOCOP: GEnetic algorithm for Numerical Optimization for COnstrained Problems

areas/genetic/ga/systems/michal/genocop/
This directory contains the GENOCOP genetic algorithm numerical optimization package by Zbigniew Michalewicz, as described in his book. GENOCOP (GEnetic algorithm for Numerical Optimization for COnstrained Problems) optimizes functions with any number of linear constraints (equalities and inequalities)
