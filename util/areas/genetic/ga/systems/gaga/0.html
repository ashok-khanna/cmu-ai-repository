GAGA: A Genetic Algorithm for General Application

areas/genetic/ga/systems/gaga/
This directory contains GAGA, a genetic algorithm for general application. It is suitable for minimization of 'difficult' cost functions. Originally written in Pascal by Ian Poole, it was rewritten in C by Jon Crowcroft.
