mGA: C and Common Lisp implementations of a messy GA

areas/genetic/ga/systems/messy_ga/
This directory contains mGA1.0. mGA is a C implementation of a messy GA as described in TCGA report No. 91008. Messy GAs overcome the linkage problem of simple genetic algorithms by combining variable-length strings, GENE expression, messy operators, and a nonhomogeneous phasing of evolutionary processing. Results on a number of difficult deceptive test functions have been encouraging with the messy GA always finding global optima in a polynomial number of function evaluations. A Common Lisp version, as documented in TCGA report No. 90004, is also included.
