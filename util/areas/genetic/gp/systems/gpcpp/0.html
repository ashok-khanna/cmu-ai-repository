GPC++: Genetic Programming in C++

areas/genetic/gp/systems/gpcpp/
This directory contains GPCplus (Genetic Programming in C++). Ramped half and half and tournament selection are implemented along with standard GP components. Project and make files are also included. A Santa Fe trail module is included which works. DOS and UNIX implementations.
