GPSTEPS: Genetic Programming SmallTalk Easy Prototyping System

areas/genetic/gp/systems/gpsteps/
GPSTEPS (Genetic Programming SmallTalk Easy Prototyping System) is a system for experimenting with Genetic Programming. The Smalltalk source code is provided in addition to an executable (MSDOS Windows 3.1) version for the Artificial Ant problem.
