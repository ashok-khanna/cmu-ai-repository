Hodge: C implementation of Gerhard and Schuster's hodge-podge machine.

areas/ca/systems/hodge/
HODGE-C is a (mostly ANSI) C language implementation of Gerhard and Schuster's hodge-podge machine. It implements a class of cellular automata, that resemble very closely autocatalytic chemical reactions, like for example, the Belousov-Zhabotinskii (BZ) reaction.
