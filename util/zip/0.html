ZIP: Compression and Archiving Program

util/zip/
There are numerous programs that can unpack files with a ".zip" extension (LZ77 with hashing, plus secondary static Huffman coding on a block basis). We've included a selection, for several architectures. Extensions: zip zipped archive (Info-ZIP format)
