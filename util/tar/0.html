Archiving Program

util/tar/
This directory contains Macintosh and IBM implementations of tar. The GNU implementation of tar is in a separate directory. Implementations include detar (1.1, 25-JAN-88), extar (1.0, 8-JUN-91, includes Unix to DOS name conversion), ltar (3, 7-FEB-92, with filename mapping), pdtar (14-JUN-88), tar4dos (21-SEP-92). Also included is a DOS/UNIX implementation of tar with compress, zip, gzip, and hardware support (316us, 16-AUG-94). File extensions include tar "Tape ARchive" files
