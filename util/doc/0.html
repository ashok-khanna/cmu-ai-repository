This directory contains helpful information about using FTP repositories: bitftp.txt Help file returned from BITFTP after sending a message with only the word help in the message body to bitftp@pucc (bitnet) or bitftp.pucc.princeton.edu (internet). ftpmail.txt Help file returned from FTPmail after sending a message with only the word help in the message body to ftpmail@decwrl.dec.com. example.txt Example of using FTP to access the CMU AI Repository. howtoftp.txt RFC 1635 "How to Use Anonymous FTP"
