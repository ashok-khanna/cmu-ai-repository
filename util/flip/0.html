FLIP: Does newlines conversions between UNIX and MS-DOS

util/flip/
Flip is a file interchange program that converts text file formats between UNIX and MS-DOS. It converts lines ending with carriage-return (CR) and linefeed (LF) to lines ending with just linefeed, or vice versa. Executables for MS-DOS and Windows have been included.
