ProTcl: Prolog interface to Tcl/Tk.

lang/prolog/gui/protcl/
ProTcl (pronounced pro-tickle) is a Prolog interface to Tcl/Tk. It has a very simple interface, which just allows one to evaluate Tcl expressions from Prolog. 
See Also: 

   http://www.ecrc.de/eclipse/html/protcl.html

   ecrc.de:/pub/eclipse/progs/protcl/
Version: 1.3 (13-SEP-94) Ports: ProTcl has been tested on ECLiPSe and SICStus, on a Sparc 10. It should also work with Quintus Prolog.
