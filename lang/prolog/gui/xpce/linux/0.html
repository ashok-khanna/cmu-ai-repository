Demo version of XPCE/SWI-Prolog for Linux

lang/prolog/gui/xpce/linux/
This directory contains a demo version (binaries) of XPCE/SWI-Prolog for PC/Linux (i386). XPCE is an object-oriented X-window interface toolkit for symbolic programming languages.
