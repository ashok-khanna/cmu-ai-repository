
Ce directory contient les versions executables sun3 et sun4 de PrologIIR.
PrologIIR est un interpreteur prolog compatible avec PrologII de marseille.
Il a ete ecrit par l'equipe MALI de l'INRIA Rennes.

Cet interpreteur est mis dans le domaine publique par l'INRIA avec l'accord
de PrologIA Marseille.

Vous trouverez ici trois fichiers :

  --> "A_LIRE"
         Le fichier que vous etes en train de lire.

  --> PrologIIR.tar
         Fichier "tar" contenant la distribution PrologIIR (executables
         sun3 et sun4).

  --> PrologIIR.tar.Z
         Version compressee (commande "compress") du fichier "PrologIIR.tar".



Que faire lorsque vous avez recupere le fichier "PrologIIR.tar" (ou
"PrologIIR.tar.Z" si vous avez la commande "uncompress") :

  1) creer un repertoire "PrologIIR" et se deplacer sous ce repertoire:

       mkdir PrologIIR
       cd PrologIIR

  2) deplacer le fichier "tar" "PrologIIR.tar" ou "PrologIIR.tar.Z"
     sous le repertoire "PrologIIR":

       mv <ou-ce-trouve-le-fichier>/PrologIIR.tar .

  3) si vous avez recupere la version "compressee" du fichier "tar",
     le decompresser:

       uncompress PrologIIR.tar

  4) extraire le contenu du fichier "tar":

       tar xvpf PrologIIR.tar

  5) suivre les instructions contenues dans le fichier README extrait.
