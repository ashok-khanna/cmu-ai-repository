WAMCC: WAM-based Prolog to C compiler

lang/prolog/impl/prolog/wamcc/
WAMCC is a WAM-based Prolog to C compiler. It conforms more or less to the Edinburgh standard, and includes most of the usual built-in predicates, a top-level, a Prolog debugger and a WAM debugger. WAMCC is designed to be easily extended (see clp(FD), for example). WAMCC's speed is halfway between SICStus emulated and SICStus native code on a Sparc (1.5 times faster and 1.5 times slower, respectively). Examples include a calendar program and the 8-queens problem.
