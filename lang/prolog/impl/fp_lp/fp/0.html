FP: A C implementation of John Backus' "FP" language.

lang/prolog/impl/fp_lp/fp/
This directory contains a C implementation of John Backus' "FP" functional programming language. Includes examples of bubble sort, discrete Fourier transform, prime number generation, and regression testing.
Version: 5-JAN-1990 Requires: C Ports: HP-UX on RISC and 68K machines. Copying: Public domain.
