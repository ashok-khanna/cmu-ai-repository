
This directory contains the latest release of the Goedel system in
tar-compressed format.  There are four files in the directory:

	goedel-src.tar.Z
	goedel-sage.tar.Z
	goedel-sparc1.tar.Z
	goedel-sparc2.tar.Z
	goedel-sparc3.tar.Z
	goedel-linux.tar.gz

goedel-src.tar.Z contains:
	1. Source code of the Bristol Goedel system,
	2. A user manual and installation instructions,
	3. A paper on ground representation in Goedel,
	4. The Goedel library, and
	5. Example programs.

goedel-sage.tar.Z contains:
	1. Source code of SAGE (Self-Applicable Goedel partial Evaluator)
	   by Corin Gurr, and
	2. SAGE user manual.

goedel-sparc1.tar.Z, goedel-sparc2.tar.Z, and goedel-sparc3.tar.Z
	together contain a stand-alone executable Goedel system for
	SPARCstations.

goedel-linux.tar.gz contains a stand-alone executable Goedel system for
        IBM PC compatibles running Linux 0.99.15 (or later).


If you have SICStus Prolog version 2.1 #6 (or later), the file
goedel-src.tar.Z should be sufficient for you to set up a complete
Goedel system.  However, if you do not have SICStus Prolog but want to
run Goedel on a SPARCStation, you will also need to obtain the files:
	goedel-sparc1.tar.Z
	goedel-sparc2.tar.Z
	goedel-sparc3.tar.Z
If you want to run Goedel on an IBM compatible running Linux, you need
the file
	goedel-linux.tar.gz.
