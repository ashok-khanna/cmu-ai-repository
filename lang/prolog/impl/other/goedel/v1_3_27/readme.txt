
This directory contains the latest release of the Goedel system in
tar-compressed format.  There are four files in the directory:

	goedel_src.tar.Z
	goedel_runtime1.tar.Z
	goedel_runtime2.tar.Z
	goedel_runtime3.tar.Z

goedel_src.tar.Z contains:
	1. A draft book on the Goedel language,
	2. Source code of the Goedel system,
	3. A user manual and installation instructions,
	4. The Goedel library, and
	5. Example programs.

goedel_runtime1.tar.Z, goedel_runtime2.tar.Z, and goedel_runtime3.tar.Z
	together contain a stand-alone executable Goedel system.


If you have SICStus Prolog version 2.1 #6 (or later), the file goedel_src.tar.Z
should be sufficient for you to set up a complete Goedel system.  However,
if you do not have SICStus Prolog, you will also need to obtain the files:

goedel_runtime1.tar.Z,  goedel_runtime2.tar.Z, and  goedel_runtime3.tar.Z,

which together contain an executable Goedel system for a SPARCstation.
