HO_UNIF: Higher Order E-Unification

lang/prolog/impl/other/ho_unif/
This directory contains an experimental implementation of of higher order E-Unification developed at the FZI, Karlsruhe. It implements the modular higher order E-Unification of Nipkow, Qian and Wang with optimizations described in Franz Weber's PhD thesis.
