EDCG: Extended DCG notation for Prolog

lang/prolog/code/tools/dcg/
Extended DCG is a preprocessor for Prolog that simplifies purely applicative programming. It allows Prolog functions to have an arbitrary number of arguments without increasing the size of the source code.
