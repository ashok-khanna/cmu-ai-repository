Prolog Consultant: Prolog programming consultant

lang/prolog/code/tools/consult/
This program is intended to serve as a "intelligent consultant" for PROLOG programmers to turn to when encountering some impasse in a programming project. The program is based on the "Eliza" program, but it designed to provide comments that might foster the user to "solve" his or her own problem.
