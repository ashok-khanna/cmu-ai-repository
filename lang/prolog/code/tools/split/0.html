SPLIT: Prolog file separator.

lang/prolog/code/tools/split/
SPLIT allows one to separate text files which have been packed together into a larger file. (SPLIT was written by Jocelyn Paine back when the files in her Prolog Library were packed in just such a manner.) The subfiles must have the following structure:
Zero or more lines of text
