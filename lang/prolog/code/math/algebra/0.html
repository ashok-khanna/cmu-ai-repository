Algebra: Computer algebra system implemented in Prolog

lang/prolog/code/math/algebra/
This directory contains Sergio Vaghi's computer algebra system. It includes Prolog programs for symbolic differentiation of algebraic functions and for symbolic simplification of algebraic functions. The programs run under PD Prolog version 1.90.
