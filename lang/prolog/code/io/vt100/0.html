VT100 VT52: Prolog cursor-addressing predicates.

lang/prolog/code/io/vt100/
This package contains Prolog cursor-addressing predicates. There are two sets of predicates, one for vt100s and one for vt52s. Operations supported include move to X,Y clear a line or page set inverse or normal video
