Explicit-Renaming Based Macros: Proposals and implementations for macros in Scheme.

lang/scheme/doc/macros/willc/
This directory contains William Clinger's proposal for high-level macros based on explicit renaming, and copies of the proposal [2,3].
Version: Implementation 13-APR-93; Papers 21-FEB-92 Ports: Tested in MacScheme, Gambit, and Chez Scheme. Copying: Copyright (c) 1992 William Clinger Use, copying, and redistribution permitted.
