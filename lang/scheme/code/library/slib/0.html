SLIB: Standard Scheme Library

lang/scheme/code/library/slib/
SLIB (Standard Scheme Library) is a portable Scheme library which is intended to provide compatibility and utility functions for all the standard Scheme implementations. SLIB conforms to R4RS and IEEE P1178. Answers to frequently asked questions about SLIB may be found in the file slib.faq.
