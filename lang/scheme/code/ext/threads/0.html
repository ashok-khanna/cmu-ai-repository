PCALL: Scheme code to emulate FUTURE and PCALL.

lang/scheme/code/ext/threads/
PCALL is a multi-threaded kernel that provides a pcall facility, an implementation of futures and suspensions, and a high-level control operator that allows the definition of partial continuations.
