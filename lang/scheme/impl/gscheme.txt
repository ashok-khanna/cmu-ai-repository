As I have receive a lot of mail
(well, 2 is sometimes a lot), here are
some informations about GScheme.

GScheme is a Scheme-like Lisp system.
It is not public domain.
A demo version for the sun4 is available
from the ftp :
	ftp.emse.fr
in
	/pub/glisp/progs/sun4
Some documentation in french is available
in
	/pub/glisp/doc
In the same directory you can find a
lisp-like version of GScheme, GLisp,
a C-like version, CTalk, and a FP-like
version, FP.
