SchemeTeX: Simple support for literate programming in Scheme.

lang/scheme/util/scm_tex/
SchemeTeX is a Unix filter that translates schemeTeX source into LaTeX source. Originally developed for the Scheme dialect of Lisp, it can easily be used with most other dialects. This is not the original distribution, but a version modified by Pertti Kellom\"aki to work with GNU TeXinfo format.
