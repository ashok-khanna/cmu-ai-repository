SCIX: Scheme interface to the X Window System

lang/scheme/gui/scix/
SCIX is an object-oriented interface to the X Window System for Scheme. Documentation is included in the distribution. HWOOPS (Heavy-Weight OOPS) is included as part of the SCIX distribution.
