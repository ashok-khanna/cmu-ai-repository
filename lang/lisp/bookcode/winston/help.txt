IF YOU WISH TO TRANSFER THE SOFTWARE:

Currently available software is written in Common Lisp.  To obtain
it, please proceed as follows:

1. Create a directory where you want the software to reside.

2. Type the following:

ftp ftp.ai.mit.edu

3. Login as anonymous.

4. Use your complete email address as your password.

5. Type the following to change to the directory where the software resides:

   cd /pub/ai3

6. Use lcd to change to the directory where you want to put the software.

7a. If you want to transfer all the software, type the following:

    mget *

7b. If you want to transfer a directory of the software, type the following:

    get readme.txt

Be sure to read the file readme.txt before you start to use any
of the programs.

IF YOUR COMPUTER DOES NOT SUPPORT FTP:

You can obtain the software from Gold Hill Computers (617/621-3300).

IF YOU WISH TO OBTAIN A DIRECTORY OF FREE PROGRAMS:

Send a message to ai3@ai.mit.edu with the word ``programs'' in
the subject line.  Your message will be answered by an automatic
message handler.

IF YOU WISH TO CONTRIBUTE TO A DIRECTORY OF FREE PROGRAMS:

Please send a message to ai3-contributions@ai.mit.edu, along with a
short description of what you want to contribute, the language
your contribution is written in, and how your contribution can be
obtained.

IF YOU WISH TO OBTAIN THE LIST OF KNOWN BUGS IN THE TEXT OR SOFTWARE:

Send a message to ai3@ai.mit.edu with the words ``text bugs'' or
``software bugs'' in the subject line.  Your message will be
answered by an automatic message handler.

IF YOU WISH TO REPORT A BUG IN THE TEXT OR THE SOFTWARE:

First, please obtain the list of known bugs in the text or
software, as described above, and check to see if your bug has
been reported previously.  Then, if it is a newly discovered bug,
please send a message to ai3-bugs@ai.mit.edu
