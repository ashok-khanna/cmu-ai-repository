Edebug: Source level debugger for Emacs Lisp.

lang/lisp/util/emacs/edebug/
This directory contains Edebug, a debugger for Emacs Lisp, and some utilities for Common Lisp debugging (Dave Gillespie's version of cl.el). Also includes cl-read.el, a CL readmacro package for Emacs Lisp.
