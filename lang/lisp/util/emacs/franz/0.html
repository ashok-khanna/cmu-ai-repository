ELI and CLMAN: Franz's Emacs-Lisp interface

lang/lisp/util/emacs/franz/
This directory contains ELI, Franz Inc.'s GNU-Emacs/Lisp interface. The interface consists of the following files: eli_2016.tgz (was eli-2.0.16.tar.gz) The source code to the interface. clman_v2.tgz (was clman-4.1-v2.tar.gz) clmanclm.tgz (was clman-4.1-v2-clim2.tar.gz) An online Common Lisp reference manual is available by license from Franz Inc. Contact info@franz.com for more information.
