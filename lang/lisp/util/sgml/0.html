SGML2Lisp: SGML output formatting tool

lang/lisp/util/sgml/
Generates a lisp program that acts as a filter in converting SGML text to any user-specified format. The generated converter operates on the output of the SGML parser SGMLS (copyleft by J. Clarke) and performs the same task as SGMLSASP.
