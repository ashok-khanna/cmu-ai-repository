CLtL2-TeX: TeX style macros for formatting Lisp code.

lang/lisp/util/tex/
TeX macros used for formatting Guy Steele's "Common Lisp the Language" first and second editions. Useful for formatting Lisp code. Also included in the distribution is a simple shell script by Nicolas Rouquette that is useful for including Lisp code in TeX files.
