This directory contains the Atari versions of CLISP,
a full Common Lisp implementation for the Atari-ST.

There are two versions of CLISP:
* The original one, written in 68000 assembly language.
  Development stopped in 1991.
  This one is fast, consumes very few memory, but may not be up to date.
* The one rewritten in C. Still maintained.
  Somewhat slower, consumes more memory, more suitable for big packages
  like PCL.
