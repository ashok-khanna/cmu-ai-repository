This directory contains executable versions of XLISP-STAT for the
Macintosh, sources for the Macintosh and UNIX versions, and LaTeX
files for a tutorial introductions to the system.

The Mac distribution for Version 2.1 consists of two 800K Disks of
binaries and executables in the files XLISP-STAT2.1Rx.Disk1.sit.Hqx
and XLISP-STAT2.1Rx.Disk2.sit.Hqx, with Rx denoting the current
release number.

The UNIX distribution for Version 2.1 is in the compressed tar file
xlispstat2.1Rx.tar.Z, again with Rx denoting the current release
number.
