Code related to Lisp PACKAGE definitions

lang/lisp/code/ext/packages/
This directory contains code related to Common Lisp packages. DEFPACKAGE defines a portable implementation of the Common Lisp function of the same name. MAKE-DEFPACKAGE creates a defpackage form that could recreate the current state of a package, more or less.
