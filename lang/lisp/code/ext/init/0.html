Initializations: Like the initializations features of Lisp machines.

lang/lisp/code/ext/init/
Mimics the initializations features of the TI Explorer and Symbolics Lisp machines. Provides for functions to be run when the world is first booted, before and after GC, before a non-checkpointed disk save, and after a non-checkpointed world is first booted.
