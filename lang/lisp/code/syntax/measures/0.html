MEASURES: Handles engineering numbers and measures in Lisp.

lang/lisp/code/syntax/measures/
MEASURES is a system to handle engineering numbers and measures in Common Lisp. Some documentation can be found in the file measures.doc and examples in measures-example.lisp in the distribution.
