Matcher: Regular expression style pattern matcher.

lang/lisp/code/match/matcher/
Simple regular expression style matching of a pattern against a list expression. Note that this is not a full regular expression matcher. Examples of use include: (match-p '((:star :wild) b :wild d) '(a b c d)) ==> t (match-p '(b (:star :wild) d (:star :wild)) '(a b c d)) ==> nil (match-p '(:wild b :wild d) '(a b c d)) ==> t
