EW: Express Windows

lang/lisp/gui/ew/
EW (Express Windows) is intended to mimic Symbolics' Dynamic Windows user and programmer interfaces. Note: EW is no longer under active development. The original version is in the original/ subdirectory. The Goodwill version is in the goodwill/ subdirectory. An archive of mail to the mailing list is kept in the mail/ subdirectory.
