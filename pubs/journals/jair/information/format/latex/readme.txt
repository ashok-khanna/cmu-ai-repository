This is a subdirectory of JAIR's format directory containing the LaTex
(and BibTex) style file. The format directory contains general
instructions for formatting, and the files here are specifically for
LaTex.

Files in this directory include:
   README            this file
   jair.sty          Latex style file for JAIR format
   sample.tex        Sample latex file
   sample.bib        Sample bibtex file
   sample.ps         Sample PostScript file
   theapa.bst        Latex bibliography style file
   theapa.sty        Latex style file supporting theapa bibliography style
