IFR_Fuzzy: Fuzzy control toolbox for MATLAB.

areas/fuzzy/systems/ifr/
This directory contains IFR_fuzzy, an easy to use fuzzy-control toolbox for MATLAB. Fuzzy variables and rules can be entered and edited with M-files, or through GUI-tools (Matlab Version 4.x). Several arithmetic operators on fuzzy sets, fuzzification and defuzzification algorithms are provided. Computationally extensive algorithms are implemented as Cmex functions.
