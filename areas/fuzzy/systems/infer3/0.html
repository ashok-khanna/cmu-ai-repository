INFER: General Purpose Fuzzy Inference Library

areas/fuzzy/systems/infer3/
This directory contains a "General Purpose Fuzzy Reasoning Library" developed at the University of Tokyo. The program is in C, with English comments, but the documentation is in Japanese. This is a set of English-language documentation for that library written by John Nagle.
