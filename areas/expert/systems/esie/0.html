ESIE and Logic-Line: Expert System Shell and AI Retrieval/Correlation Tool.

areas/expert/systems/esie/
ESIE (Expert System Inference Engine) is an expert system shell for personal computers. Logic Line is an AI retrieval and correlation tool. It can be used to find information in an ASCII file or database using keywords. One can using it to create rules for ESIE.
