CLIPSDLL: Creates a CLIPS dynamic link library for MsWindows.

areas/expert/systems/clips/contrib/clipsdll/
CLIPSDLL contains a set of modified and added files to the standard CLIPS distribution that can be used to create a CLIPS 5.1 Dynamic Link Library for Microsoft Windows. This version also includes a working CLIPS DLL, ready for use in applications that do not require a custom UserFunctions() function.
