******************************************************************
		ANNOUNCEMENT
		SLUG4 GOES COMMERCIAL
******************************************************************

Firstly, thanks to all those who have used the last public domain version, 
SLUG3, as found in NEURAL22.ZIP all over the place(more than 30 sites, at my 
last archie count).

SLUG4 is now a commercial product.  It includes a 52 page manual, the entire 
object oriented neural networks toolkit used to build SLUG4, plus all the 
Borland Pascal source code.

It costs $59, plus $10 shipping.

SLUG4 will automatically scale your data, implements epoch based training, 
allows you to choose transfer functions and randomizes your data on the fly 
during training. 

The toolkit provides a flexible and very easy to use paradigm for the 
construction of ANY neural net, and includes backprop net objects.

It also contains :
A neuron object
A croupier object (for random 'dealing' of data)
A dynamic matrix object (used for weights)
A dynamic vector object (used for neuron 'slabs' etc)
Code for reading and constructing Brainmaker (California Scientific) backprop 
nets.

You can order from Southern Scientific using your Mastercard or VISA card, via 
FAX (27-21-788-6613). Remember to specify the expiry date.
