RCC: Recurrent Cascade-Correlation Learning Algorithm in Lisp and C

areas/neural/systems/rcc/
This directory contains Lisp and C implementations of the recurrent cascade-correlation learning algorithm. The original Lisp version was written by Scott Fahlman. It was then ported to C by Scott Crowder. This directory contains the C version modified by Conor Doherty.
