CASCOR: Lisp and C implementations of Cascade Correlation

areas/neural/systems/cascor/
This directory contains C and Lisp implementations of the Cascade Correlation Algorithm. The Lisp code was written by Scott Fahlman, then ported to C by Scott Crowder. More recent C versions have been re-engineered by Matt White. Features have been added to allow data sets from the CMU learning benchmark database to run on this system.
