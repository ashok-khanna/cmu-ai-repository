FM_2D: 2-d feature mapping for 2-d input vectors

areas/neural/systems/utexas/fm_2d/
This package forms a 2-d feature mapping for uniformly distributed 2-D input vectors in the unit square x,y e [0,1] (includes fancy graphics displays for HP workstations).
