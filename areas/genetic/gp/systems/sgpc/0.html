SGPC: Simple Genetic Programming in C

areas/genetic/gp/systems/sgpc/
This directory contains SGPC (Simple Genetic Programming in C), a simple Koza and Rice workalike system written in C. It is a C program that writes the same kind of Lisp programs as the Koza system, but is very portable and somewhat faster.
