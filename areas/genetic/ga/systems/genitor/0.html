GENITOR: Modular GA package with floating-point support.

areas/genetic/ga/systems/genitor/
This directory contains GENITOR. Genitor is a modular GA package containing examples for floating-point, integer, and binary representations. Its features include many sequencing operators as well as subpopulation modeling.
