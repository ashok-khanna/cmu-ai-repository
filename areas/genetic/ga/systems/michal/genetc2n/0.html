Genetic-2N: Optimization package for the nonlinear transportation problem

areas/genetic/ga/systems/michal/genetc2n/
This directory contains the Genetic-2N genetic algorithm numerical optimization package by Zbigniew Michalewicz, as described in his book. Genetic-2N is an optimization package for the nonlinear transportation problem (minimization of the transportation cost).
