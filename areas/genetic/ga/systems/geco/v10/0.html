GECO: Genetic Evolution through Combination of Objects

areas/genetic/ga/systems/geco/v10/
This directory contains GECO, an extendible object-oriented toolbox for constructing genetic algorithms. It provides a set of extensible classes and methods designed for generality. Some simple examples are also provided to illustrate the intended use.
