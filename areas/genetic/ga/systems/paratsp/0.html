ParaTSP: Parallel GA with Simulated Annealing to solve TSP's

areas/genetic/ga/systems/paratsp/
ParaTSP is a software package for solving Traveling Salesman Problems (TSP) using genetic algorithms (GA) and simulated annealing (SA). ParaTSP is based on Baeck's GeneSyS.
