GENALG: Genetic Algorithm package written in Pascal

areas/genetic/ga/systems/genalg/
This directory contains GENALG, a program that demonstrates the operation of genetic algorithms. From a randomly generated population of real numbers, it attempts to find a number arbitrarily close to a goal real number.
