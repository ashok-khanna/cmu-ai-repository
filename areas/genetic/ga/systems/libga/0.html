LibGA: Library of GA routines written in C.

areas/genetic/ga/systems/libga/
This directory contains LibGA, a library of genetic algorithm routines written in C. It is simple and easy to use, with many knobs to turn. Both generational and steady state models supported. Most GA parameters can be set or changed via configuration file, with no need to recompile. Function pointers are used for the GENETIC OPERATORs, so they can easily be manipulated on the fly. Several genetic operators are supplied and it is easy to add more.
