Soar: Integrated Agent Architecture

areas/planning/systems/soar/soar6/
This directory contains the C version of Soar. Soar is an integrated agent architecture that supports learning through chunking. Included are soar-mode for GNU Emacs, the Soar Development Environment (SDE), Dismal, and SimTime.
