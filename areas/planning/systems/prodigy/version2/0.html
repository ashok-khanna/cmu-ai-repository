Prodigy: Integrated Planning and Learning System

areas/planning/systems/prodigy/version2/
This directory contains the Prodigy planning and learning system. Domains include Blocksworld, Gridworld (3d Blocksworld), Logic, Matrix Algebra (Gaussian Elimination), MultiRobot, R1 (VAX configuration), Rocket, Schedworld (Machine Shop Scheduling), STRIPS World, Telescope Mirrors, Path Planning, Jupiter, 8-Puzzle, Monkey and Bananas, and Towers of Hanoi. The additional Prodigy domains in domains.tgz were provided by Oren Etzioni .
Version: 2.11 (2-OCT-90) Requires: Common Lisp Ports: CMU CL, MCL, Lucid CL, Allegro CL Copying: If you use Prodigy regularly, please sign and return a copy of the release form (release.ps).
