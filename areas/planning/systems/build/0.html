BUILD: A blocks-world planner

areas/planning/systems/build/
This directory contains a Common Lisp version of Scott Falhman's Build, a blocks-world planner. It is a classical recursive-descent planner, and is one of the first solid geometric modelers. It was converted from CONNIVER to Common Lisp by John Nagle.
Version: 5-DEC-88 Requires: Common Lisp Ports: KCL, Ibuki CL, Lucid CL CD-ROM: Prime Time Freeware for AI, Issue 1-1 Author(s): John Nagle Keywords: Authors!Fahlman, Authors!Nagle, BUILD, Blocksworld, Lisp!Code, Planning, Recursive-Descent Planning, Solid Geometric Modeling References: Fahlman, S.E., "A Planning System for Robot Construction Tasks", Artificial Intelligence, 5:1-49, 1974.
