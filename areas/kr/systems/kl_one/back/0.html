BACK: Berlin Advanced Computational Knowledge Representation System

areas/kr/systems/kl_one/back/
BACK supports reasoning based on terminological logics (KL-ONE), supporting inheritance, consistency checking, cycle detection, classification, completion of partial descriptions, role inferences, abox revision, and extended query answering. The main system components are TBox (KB scheme), ABox (KB assertions), IBox (extensional implications), and the uniform access language. The system is currently being re-implemented in C++. (NSL, 57-59 rue Lhomond, F-75005 Paris; e-mail: back@nsl.fr).
