ZENO: Temporal Planning with Continuous Change

areas/reasonng/temporal/zeno/
This directory contains the ZENO temporal planner. It is a least commitment planner that handles actions occurring over extended intervals of time. ZENO supports deadline goals, metric preconditions, metric effects, and continuous change. It is sound and complete.
