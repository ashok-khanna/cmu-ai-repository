WireWorld: Cellular Automaton

areas/ca/systems/wirewrld/
Wire World is a cellular automaton for building logical circuits based on simple state-changes. The program includes a simple circuit editor and comes with example logic gates. Based in part on code posted by Kevin Dahlhausen to comp.theory.cell-automata.
