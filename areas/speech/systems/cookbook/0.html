Cookbook: Speech processing functions in C.

areas/speech/systems/cookbook/
This directory contains cookbook of C functions that are often used in speech processing. These currently include LPC, FFT, DTW, windowing, and supporting code such as file reading and command line argument passing. Example applications include a simple DTW based recognizer and a simple Viterbi based pitch tracker. This package is still under active development.
