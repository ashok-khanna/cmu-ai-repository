areas/nlp/corpora/wordnet/
This directory contains a copy of the WordNet README file. WordNet is an online lexical reference system whose design inspired by current psycholinguistic theories of human lexical memory. English nouns, verbs, and adjectives are organized into synonym sets, each representing one underlying lexical concept. Different relations link the sets.
