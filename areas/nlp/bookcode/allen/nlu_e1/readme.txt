There are four projects that make up this software: (1) a simple
recursive transition network, (2) an augmented transition network, (3)
an ELIZA program, and (4) a logic-based parser.  Each project is
composed of several files including a documentation file.  The user
should follow the instructions in the documentation files when running
the corresponding programs.
