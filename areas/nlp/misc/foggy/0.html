Foggy: Pseudotext Generator

areas/nlp/misc/foggy/
This directory contains Foggy, an extremely simple, "one-from-column-A and one-from-column-B" pseudotext generator, with TurboPascal V.5.5 source code and modules covering Software Engineering, Folklore Research, and Government and Binding Theory.
