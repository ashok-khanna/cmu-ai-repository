This directory contains the Alvey Natural Language Tools (ANLT). To find
out more, fetch the files

basic       basic description of the ANLT and distribution arrangements
licence     a machine-readable specimen licence agreement
instruct    instructions on how to FTP the ANLT itself

The latest release of the ANLT is release 4. If you have paid for release 3
or an upgrade to it, you are entitled to FTP release 4 at no extra charge.
The release is contained in the files anlt4.tar.Zb and anlt4.tar.Zl; the
decryption key is the same as the one you were given for release 3.

See the file notes4.txt in this directory for technical information on
release 4 that has been added since the release date.
