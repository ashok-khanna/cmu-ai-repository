Conceptual Graph Tools

areas/nlp/misc/cg/
This directory contains a variety of tools for working with conceptual graphs (e.g., Sowa's Conceptual Structures Theory). It includes a simple C/YACC parser for the linear form of a Conceptual Graph, a conceptual graph editor, and more.
