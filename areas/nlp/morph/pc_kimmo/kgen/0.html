KGEN: Rule compiler for PC-Kimmo

areas/nlp/morph/pc_kimmo/kgen/
KGEN is a rule compiler for PC-Kimmo. It translates two-level morphological rules like y:i => @:C___+:0 into a format suitable for use with Kimmo.
