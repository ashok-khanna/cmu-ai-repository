TULIP: A two level phonological formalism.

areas/nlp/morph/tulip/
This directory contains the source code for TULIP, a partial implementation of the system described in S. G. Pulman and M. R. Hepple, "A Feature Based Formalism for Two Level Phonology: A description and implementation", Computer Speech and Language. The implementation was by Hepple, based on an original version by Pulman.
