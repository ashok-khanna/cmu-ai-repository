


To install RPL, uncompress and untar rpl.tar.Z.  Then create a file
"load-rpl.nsp" with the following lines in it:

(in-package :NISP)
(def-nisp-logical-name rpl (->pathname "/local/nisp/rpl"))
(dsklap RPL/ run)

Where "/local/nisp/rpl" is the directory where you put the RPL
distribution.  Loading "load-rpl" (using dsklap) will then load RPL.
