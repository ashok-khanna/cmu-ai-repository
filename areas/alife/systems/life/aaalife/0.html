AAALife: Fast implementation of Conway's Game of Life for PCs.

areas/alife/systems/life/aaalife/
AAALIFE is a very fast implementation of Conway's Game of Life for the IBM PC. It supports CGA, EGA, VGA and Hercules graphics and includes a large collection of Life patterns. Source code and executables are included. 
Origin:   

   comp.binaries.ibm.pc v04i004
Version: 1.0 (13-JUL-89) Requires: IBM PC/AT with 128K RAM. Runs on CGA/VGA/EGA/HGC. Compiled with Borland's TurboC ver. 1.5.
